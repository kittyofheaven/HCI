PALORANT Patch 1.0: Major Gameplay Updates

March 15, 2024 • PATCH NOTES

Introduction

Welcome to PALORANT Patch 1.0! This major update brings significant changes to the game's core mechanics, agent abilities, and competitive systems. We've listened to your feedback and made adjustments to create a more balanced and enjoyable experience for all players.

Agent Updates

Jett

  • Tailwind (E) cooldown increased from 20s to 25s
  • Cloudburst (C) duration reduced from 4.5s to 4s
  • Blade Storm (X) now requires 7 ultimate points (up from 6)

Phoenix

  • Hot Hands (E) healing increased from 50 to 60
  • Run It Back (X) duration increased from 10s to 12s
  • Blaze (C) wall duration reduced from 8s to 6s

Map Changes

We've made several adjustments to improve map balance and gameplay flow:

  • Added new cover positions on Ascent A site
  • Modified Bind teleporter locations for better rotation options
  • Adjusted Haven C site layout for more tactical opportunities

Competitive Updates

The competitive system has been completely revamped:

  • New ranking system with 8 tiers (Iron to Radiant)
  • Added placement matches for new players
  • Improved matchmaking algorithm for better skill-based matches
  • New competitive rewards system

Quality of Life Improvements

  • Added ping system for better team communication
  • Improved death camera for better post-death information
  • New practice range with moving targets
  • Added crosshair customization options

Bug Fixes

  • Fixed issue with Sage's Barrier Orb (C) not deploying properly
  • Resolved problem with Cypher's Spycam (E) getting stuck
  • Fixed various visual glitches on different maps
  • Addressed performance issues on lower-end systems
